# Business Intelligence Queries

## Overview

A collection of production-ready SQL queries that answer real business questions against the e-commerce schema. These demonstrate proficiency with CTEs, window functions, aggregations, and analytical patterns commonly needed in data-driven organizations.

## Query Categories

| Category | Description |
|---|---|
| Revenue Reports | Daily, monthly, and category-level revenue breakdowns |
| Customer Analytics | Top spenders, cohort analysis, RFM segmentation |
| Product Performance | Best sellers, review analysis, cross-sell opportunities |
| Inventory Management | Low stock alerts, reorder recommendations |
| Growth Metrics | Year-over-year comparisons, moving averages, trends |

## SQL Features Demonstrated

- Common Table Expressions (CTEs) and recursive CTEs
- Window functions: `ROW_NUMBER`, `RANK`, `LAG`, `SUM OVER`, `AVG OVER`
- Date manipulation and interval arithmetic
- `CASE` expressions for business logic
- Subqueries (correlated and non-correlated)
- `COALESCE`, `NULLIF`, and conditional aggregation
- `FILTER` clause for PostgreSQL-native conditional aggregation
